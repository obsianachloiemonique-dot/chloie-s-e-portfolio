lesson 1:

css

-cascading style sheet

-used to design web pages

layers of a webpage

content layer

-the content layer is the underlying code of a page

-defines and layout the text, images, animation, sound, video and everything you want to put in a web page with the use of tags

presentation layer

-where you can define how people see your web page in a browser

-this includes the ability of the webpage to work and its appearance to the users.

behavior layer

where you have a real-time useinteraction with the web page. The interaction may be simple validation of input or filling out a form or even abig as web-based programs.

this includes the functionality of the elements in your webpage

In lesson 1, I learned about css and its purpose and benefits; css is used for designing webpages and making it look good. I also learned bout all layes of a webpage and their function in awebpage; the content layer includes the texts, image, videos, audio etc., the presentstion layer is how the users see the webpage, while the beahavior layer is the interaction between the users and your webpage, such as filling out the forms.

lesson 2:

In lesson 2, I learned about the structure of CSS and the three kinds of stylesheets; The external stylesheet is the most global one and it can affect all the pages, the embedded stylesheet is document-wide and can affect one page, while internal stylesheet override external and embedded, it is used for isolated changes to headers, paragraphs and other elements.

lesson 3:

In lesson 3, I learned about display properties; the block-level displays text vertically, while the inline-level displays text horizontally. I also learned about div and span tag and its attribute 'class'; the div tag is used for adding styles for block-level elements like paragraphs, while the span tag is used for adding styles to inline-level elements. The class attribute indicates the style to be applied.

lesson 4:

In lesson 4, I learned about classes; classes are used for formatting texts and it allows you to apply the same style to many html elements. Classes can be used for different elements and you can use more than one class to an element. I also learned that you can have classes for a specific element like p.classname, it can only be used for paragraphs.

lesson 5:

In lesson 5, I learned about css selectors; selectors are used in external stylesheets, they have an external file. Selectors have 6 types; Class selectors, Element selector or Tag selector, ID selector, Universal selector, Group selector and Attribute selector. CSS selectors are used to “find” or “select” the HTML elements that you want to style.

lesson 6:

In lesson 6, I learned about dimensions; dimension are used to manage visibility, height and width of different HTML elements. There are also Pseudo-classes arent user-defined and are used for specific HTML elements only. Example is the anchor element which has four pseudo-classes; the Link, Hover, Visited and Active which have a specific order which is LVHA.
